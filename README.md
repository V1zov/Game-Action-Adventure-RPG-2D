# Game Action Adventure RPG 2D

## 📝 Deskripsi Singkat
Game ini adalah project game pertama yang saya kembangkan. Project ini dibuat sebagai langkah awal eksplorasi saya dalam dunia *game development* sekaligus untuk menerapkan logika pemrograman dalam lingkungan yang interaktif.

Game ini memiliki genre Action dan Adventure.

## 🛠️ Teknologi yang Digunakan
* **Game Engine / Framework:** Godot 4.6
* **Bahasa Pemrograman:** GDScript
